# API

## Directories
 - [ ] Create Directory         - POST    /directory                                (name: string)
 - [ ] Get Directory            - GET     /directory/{id}
 - [ ] Get Directories          - GET     /directory

## Chats
 - [ ] Create Chat              - POST    /directory/{id}                           (name: string)
 - [ ] Get Chat                 - GET     /directory/{from id}/{chat id}
 - [ ] Get Chats                - GET     /directory/{from id}
